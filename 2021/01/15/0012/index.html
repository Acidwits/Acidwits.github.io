VS Code配置C++环境（Win10）

2021年01月15日 657 字 大概 3 分钟
  1. 1. 下载并安装VSC
  2. 2. 下载并安装MinGW
  3. 3. 配置环境变量
  4. 4. 配置环境
  • Dev-C++不好看
  • VS太大
  • VSC针不戳

下载并安装VSC

  • 官网下载对应的版本即可

  • 安装时这些全选即可

  • 安装插件cpptools

  • 其他有必要的插件也可以安装，如主题美化、图标美化、汉化等等

下载并安装MinGW

  • 打开下载网址
  • 向下翻找到这个下载

  • 解压mingw64文件夹到一个目录，这里我选择C:\Program Files

配置环境变量

  • 复制mingw64文件夹中bin文件夹的路径C:\Program Files\mingw64\bin
  • 按照图示配置环境变量

  • 测试是否配置成功，打开cmd，输入gdb -v，若显示版本则为成功

配置环境

  • 创建一个专门写C++代码的文件夹，在VSC中打开并新建一个cpp文件

  • 打开调试界面

  • 会自动生成launch.json和tasks.json，按照下边修改
{
    // 使用 IntelliSense 了解相关属性。

    "configurations": [
        {
            "name": "g++.exe - 生成和调试活动文件",
            "type": "cppdbg",

            "cwd": "C:\\Program Files\\mingw64\\bin",

            "setupCommands": [
                {
                    "description": "为 gdb 启用整齐打印",
                    "text": "-enable-pretty-printing",
                    "ignoreFailures": true
                }
            ],
            "preLaunchTask": "C/C++: g++.exe build active file"
        }
    ]
}
