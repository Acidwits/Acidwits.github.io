覆盆子（树莓派）初体验

2019年05月01日 497 字 大概 2 分钟
  1. 1. 前言
  2. 2. 材料
  3. 3. 开始瞎搞
  4. 4. Upd 2020.1.20

前言

日夜思念的鲜美多汁的树莓派总算就在眼前了呢:wink:

  • Raspberry
  • 树莓
  • 别名：覆盆子
  • 其他别名：山莓，树倭萢、山抛子、牛奶萢、撒秧萢、三月萢、四月萢、乌萢、大麦萢、泡儿刺、刺葫芦、馒头菠 （233） 、高脚波、苗子

材料

  1. 树莓派3代B+ RS版
  2. 散热片×3
  3. 树莓派3代B型/3B+ 专用电源适配器 5V 2.5A
  4. 树莓派3B/B+ 带风扇外壳
  5. 金士顿（Kingston）32GB TF(Micro SD) 存储卡 U1 C10
  6. 无线键盘鼠标套装
  7. HDMI线
  8. VGA线
  9. HDMI转VGA转换器

开始瞎搞

首先是一个盒子装着树莓派主板还有集成了多种语言的说明书
盒子那么小拿到的时候都惊到了
然后是电源，本来以为普通的microUSB线就可以的，但看到说功率不够的话会死机XDD，于是还要买专用电源
存储卡最好选16GB、32GB的，class10，容量太大不代表好，够用即可
无线键鼠emm不太好用 击键有时候没反应，敲着敲着就丢一个字母，谨慎选择
HDMI、VGA、HDMI转VGA
树莓派有一个HDMI输出口
也可以用GPIO（那一堆针脚）， 反正我不会
散热片、case和风扇都装好
无线键鼠的接收器接到树莓派上（一直以为只有俩USB接口，怕不够才买了无线键鼠，结果有4个）
装case的时候没看树莓派，直接乱拼亚克力板子 反正拼成长方体就行了 ，后来再多次拆卸过程中掰掉了几块QAQ，于是只好用502稍微固定了一下
风扇红线接4号针脚，黑线接6号针脚
有说风扇噪音大的，实测其实还可以，不靠近听的话，感觉不到的
装好之后还挺精美
卖家还送了几个夹子emm
