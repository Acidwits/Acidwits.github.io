视频线

就是用来传输视频信号的线啦
有很多种，本文介绍几种常见的类型：VGA，DVI，HDMI，DP

VGA

来自百度百科
VGA接口就是显卡上输出模拟信号的接口，VGA（Video Graphics Array）接口，也叫D-Sub接口。
VGA接口是一种D型接口，上面共有15针空，分成三排，每排五个。

也叫D-Sub口
VGA只有视频I/O，没有音频I/O
是一种比较老的线了，四种线中仅有的**只能**进行模拟信号传输的线
CRT显示器因为设计制造上的原因，只能接受模拟信号输入，所以此时VGA线是必备的
但是可以直接接收数字信号的液晶显示器出现后逐渐被取代emm，高端显卡显示器什么的应该看不到了吧(例如上边的 并非高端显卡的 RX 580就没有VGA，而是DVI×1，HDMI×1，DP×3)
在知乎上看到如果用料够好，做工过关，VGA是可以支撑到2560×1600这样的分辨率的，但平常见到的VGA大多支持到1080P

DVI

来自百度百科
DVI（Digital Visual Interface），即数字视频接口,是由1998年9月，在Intel开发者论坛上成立的数字显示工作小组（Digital Display Working Group简称DDWG）发明的一种用于高速传输数字信号的技术，有DVI-A、DVI-D和DVI-I三种不同类型的接口形式。

DVI有不同的种类

其中DVI-A、DVI-D分别只能传输模拟信号、数字信号，DVI-I则都可以传输
不同种类的性能也不同，双通道比单通道传输快，所以相同刷新率情况下，双通道的分辨率可以到更大
最大支持2K分辨率（双通道，60Hz）

HDMI

来自百度百科
高清晰度多媒体接口(High Definition Multimedia Interface，HDMI)是一种兼具高清晰数字视频和数字音频传输能力的接口标准，是适合影像传输的专用型数字化接口，其可同时传送音频和影像信号。HDMI具有体积小，传输速率高，传输带宽宽，兼容性好，能同时传输无压缩音视频信号等优点。

按照电气结构和物理形状的区别，HDMI接口可以分为Type A、Type B、Type C、Type D四种类型，其中A、C、D常见

Type C和Type D接口分别被称为mini HDMI和micro HDMI,丰富的兼容各种尺寸设备接口的支持让HDMI的推广普及顺风顺水
最新的HDMI规格(HDMI 2.1)支持一系列更高的视频分辨率、包括8K 60Hz和4K 120Hz在内的刷新频率，以及高达10K的分辨率,同时支持HDR格式，带宽增加到48Gbps

DP

来自百度百科
DisplayPort（简称DP）是一个由PC及芯片制造商联盟开发，视频电子标准协会（VESA）标准化的数字式视频接口标准，主要用于视频源与显示器等设备的连接，并也支持携带音频、USB和其他形式的数据。

也有很多种啦，比如Mini DisplayPort但是见得较少就不细说了，就是样子不太一样，小一点的接口
DP1.4最高达32.4Gbps的带宽，可支持8K 60Hz 、4K 120Hz
目前DP正在准备新的标准，预计可以达到64.8Gbps的速率
插拔时要按下卡扣，防止拉坏

总结

常见几种线的简单介绍就这样了，更深入的了解还是要自己动手吧
另外还有很多种视频线没介绍到，比如Type-C、ThunderBolt等，有兴趣可以自己查一下，兴趣是最好的老师
