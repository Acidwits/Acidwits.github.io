DIY流光溢彩效果

某天夜里突然想玩LED灯带 看这个效果不错 打算做一个
视频链接

准备材料

  1. Arduino Nano开发板带USB线
  2. WS2812B灯带 30灯 1米
  3. 驱动 要烧录的代码 Arduino IDE Prismatik软件 (天翼云访问码:a1ym)
  4. 杜邦线

注意



  • 价钱应该是裸板>滴胶>套管
  • 灯带可以自己选择灯数和长度
  • 滴胶的做这个效果应该不错 刚好可以用背面的双面胶粘到显示器后边 我用的套管
  • 我是24寸显示器 贴了两侧和上边 一米的灯带还算够用

配置

Arduino连接配置

  • Arduino要是配置出现问题完全可以问一下店铺客服
  • 首先开发板连USB到电脑 安装驱动程序CH341SER.EXE
  • 安装完成后打开设备管理器 在端口中找到开发板对应的端口并记下来

Arduino IDE配置

  • 安装Arduino IDE
  • 在IDE的菜单-工具-开发板选择Arduino Nano 处理器选择Atmega328P-old BootLoader（我的这个板子是这样 具体可以问客服） 端口选择对应的刚才记下的端口
  • 学习一下IDE的基本操作 我是看这篇文章学的
  • 总之在这步确保板子可以烧录程序 运行程序
  • 菜单-项目-加载库-管理库搜索FastLED 安装FastLED库

开发板连接灯带

  • 尽量选和原先的线颜色一样的杜邦线连接板子和灯带 方便区分
  • 白线是地线 接到开发板的某个GND针脚
  • 红线是电源线 接到某个5V针脚 GND附近就有（我这里30灯 灯太多了可能带不起来要自己加电源）
  • 绿线是信号线 接到有PWM功能的针脚上

将源码烧入开发板

  • 打开ambilight.ino 打不开可能是因为文件在中文目录下 大不了就新建一个项目把代码copy过去
  • 修改一下代码开头的NUM_LEDS和DATA_PIN 对应为灯带灯数和灯带信号线接的针脚序号
  • 烧入开发板成功就好了 会亮三下（R、G、B）
  • 没亮的话可以看一下线是否接好 或者尝试换一下信号线接的针脚

安装配置Prismatik

  • 这里是官网下载界面（打开慢的话也可以用我提供的安装包） 也有其他平台 貌似win平台只有64位
  • 安装软件
  • 可以在安装快结束时配置 安装完成之后也可以进行流光溢彩的配置
  • 注意串口要对应
  • 灯对应的位置可以一个一个调 也可以用自带的Custom进行配置
  • 具体请看图










  • 最后放个视频 大功告成
